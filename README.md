<div style="border: 3px solid #ffffff; background-color: #ffffff; color: #000000; padding: 20px; margin: 20px 0; border-radius: 8px;">
  <h2 style="margin-top: 0;">О проекте</h2>
  Простая, но концептуально сильная социально-психологическая хоррор-игра в классическом визуальном стиле PS1.<br>
  <strong>Ключевые особенности:</strong><br>
  • Визуал: low-poly, dithering, VHS-эффекты и grain<br>
  • Геймплей: walking simulator + branching dialogue<br>
  • Длительность: 20–30 минут (идеально для первой игры)
</div>

<div style="border: 3px solid #ffffff; background-color: #ffffff; color: #000000; padding: 20px; margin: 20px 0; border-radius: 8px;">

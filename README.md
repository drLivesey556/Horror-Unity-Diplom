> **О проекте**  
> Простая, но концептуально сильная социально-психологическая хоррор-игра в классическом визуальном стиле PS1.  
> 
> **Ключевые особенности:**  
> • Визуал: low-poly, dithering, VHS-эффекты и grain  
> • Геймплей: walking simulator + branching dialogue  
> • Длительность: 20–30 минут (идеально для первой игры)

> **Технологический стек**  
> • **Движок:** Unity 2022.3 LTS (Universal Render Pipeline)  
> • **Рендеринг:** URP + Shader Graph (PS1-эффекты)  
> • **Камера:** Cinemachine  
> • **Ввод:** Input System  
> • **Пост-обработка:** Volume Profile
